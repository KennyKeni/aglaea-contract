# @aglaea/contract

Shared TypeBox schemas and TypeScript types for Aglaea and Herta.

Consume this package from npm as a normal dependency, not through a workspace or
local path reference.

## Install

```sh
npm install @aglaea/contract
```

## Exports

- `@aglaea/contract`
- `@aglaea/contract/pokemon`
- `@aglaea/contract/read-domain`

## Development

```sh
npm install
npm test
npm run pack:dry-run
```

Normal releases use the GitHub trusted-publishing workflow. Use
`npm run publish:public` only for a manual publish.
